## 🔐 Proyectos de Ciberseguridad

### 🛡️ Auditoría de Seguridad Web (OWASP Top 10)
Análisis de vulnerabilidades realizado sobre una aplicación web utilizando **OWASP ZAP**.

- Identificación de vulnerabilidades relacionadas con **Content Security Policy, X-Frame-Options y Subresource Integrity**.
- Clasificación de hallazgos mediante **CWE (Common Weakness Enumeration)**.
- Elaboración de **reporte técnico de auditoría de seguridad** con evidencia, impacto y recomendaciones de mitigación.
